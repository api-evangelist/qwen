# Qwen (qwen)

Qwen is Alibaba's open-source large language model family, including Qwen2.5, Qwen3, Qwen-VL (vision), Qwen-Coder, and Qwen-Audio. APIs are exposed via Alibaba Cloud Model Studio (DashScope) with per-token pricing.
